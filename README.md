# py7zip
A powerful version of 7zip without the need for an actual installation. Just install the Python library and you're good to go!

Simply clone this repo and run ```pip install .```. Enjoy! I've also included the precompiled 7zz binary in the package so you no longer have to wait for it to compile on install. It will install much faster!
